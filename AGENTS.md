# 项目说明

这是我的个人网站项目

## 技术栈

- Astro
- TypeScript
- Tailwind CSS
- MDX
- pnpm

## 开发偏好

- 默认使用简体中文回复。
- 优先保持页面小清新、干净、轻盈、有细腻动效。
- 不要引入过重依赖，除非明确说明必要性。
- 改动前先说明计划修改哪些文件。
- 不要一次性重构整个项目。
- 保持现有项目结构，除非我明确要求调整。
- 代码注释优先使用中文。
- Tailwind 类名要保持清晰，不要堆得过度混乱。

## 命令

- 安装依赖：pnpm install
- 本地开发：pnpm dev
- 构建：pnpm build
- 预览：pnpm preview

## 操作规则

- 修改前先阅读相关文件。
- 修改后说明改了什么、为什么改、如何测试。
- 如果涉及样式修改，需要考虑移动端响应式。
- 如果涉及动画，需要注意性能，不要让页面卡顿。
