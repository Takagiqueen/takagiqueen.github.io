## 不只是写代码

写代码之外，我对视觉和设计一直有兴趣。这些东西让我做前端界面时会更在意色彩、间距和画面节奏——不只是把东西"摆上去"。

## Blender

学过一阵 Blender，建模、材质、灯光、渲染的基本流程都摸过。虽然现在不做三维了，但对光影和构图的敏感还在。做网站背景渐变、卡片阴影的时候，脑子里会自然冒出那些画面。

## Lightroom 和摄影

拍过一些照片，用 Lightroom 做后期。也学了点摄影基础：三分构图、光影方向、色彩怎么搭。这些东西在做网页配色和视觉层次的时候帮了不少忙。

## SolidWorks

专业课上学的 SolidWorks 参数化建模。虽然和 Web 开发看起来没关系，但它让我对尺寸比例和结构合理性变得比较敏感。写 CSS 布局的时候会在意对齐和间距，大概跟这个有关。

## AstrBot

给 QQ 聊天机器人接入了 AI，用的 AstrBot。过程不复杂，就是 API Key 配置、消息格式处理、指令路由这些。不算很深的技术经验，但了解了 AI 工具接入真实聊天场景的大概流程。

## 小结

跨领域的东西不需要每个都精通。保持好奇，每个领域学到的那一点审美或思维习惯，换一个地方可能刚好用上。
