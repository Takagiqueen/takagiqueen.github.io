## 不只是写代码

写代码之外，我对视觉和设计一直有兴趣。这些东西让我做前端界面时会更在意色彩、间距和画面节奏——不只是把东西"摆上去"。

## Blender

学过一阵 Blender，建模、材质、灯光、渲染的基本流程都摸过。虽然现在不做三维了，但对光影和构图的敏感还在。做网站背景渐变、卡片阴影的时候，脑海里会自然冒出那些画面。

## Lightroom 和摄影

拍过一些raw照片用 Lightroom 做后期。也看过影视飓风网课学了点摄影基础：三分构图、光影方向、色彩，光圈，白平衡，曝光怎么搭等等这些东西在做网页配色和视觉层次的时候帮了不少忙。

## SolidWorks

B站网课学了 SolidWorks 参数化建模。

## AstrBot

给 QQ 聊天机器人接入了 google本地ollama模型，用的 AstrBot。过程不复杂，就是 API Key 配置、消息格式处理、指令路由这些。不算很深的技术经验，但了解了 AI 工具接入真实聊天场景的大概流程。

## Claude code 和 Codex

能够熟练调用国外agent模型，配置skill检查开源skill是否存在挖矿行为，能够独立创建skills。个人热爱探索深度学习相关知识。

## 小结

跨领域的东西不需要每个都精通。保持好奇心，每个领域学到的那一点审美或思维习惯，换一个地方可能刚好用的上。
